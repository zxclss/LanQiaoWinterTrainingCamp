# Practice_0x01

## C - 2-C

### Problem:

蒜术师给了你一个 ${1010}$个整数的序列，要求对其重新排序。排序要求:

1. 奇数在前，偶数在后；
2. 奇数按从大到小排序；
3. 偶数按从小到大排序。

### 输入格式

输入一行，包含 ${1010}$ 个整数，彼此以一个空格分开，每个整数的范围是大于等于 ${0}$，小于等于 ${100}$。

### 输出格式

按照要求排序后输出一行，包含排序后的 ${10}$ 个整数，数与数之间以一个空格分开。

### Sample Input

```
4 7 3 13 11 12 0 47 34 98
```

### Sample Out

```
47 13 11 7 3 0 4 12 34 98
```
